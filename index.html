Tecnologias

PHP

PHP - Uma ferramenta antiga de programadores que ainda tem muito no mercado, usamos para de modo com que alguns trabalhos tenham velocidade em seu término, pela facilidade de manuseiar o banco de dados.

NodeJS

NodeJS - Uma forte plataforma de desenvolvimento, utilizando ele pra projetos mais organizados e pra projetos modernos tentamos alcançar o projeto mais bem elaborado possivel, fácil e de simples manutenção.

MongoDB

MongoDB - Um banco de dados moderno orientado a documentos e dependendo do seu projeto, pode ser a melhor escolha!

MySQL

MySQL - Um banco de dados antigo porém com enorme funcionalidade dos dias de hoje, banco de dados com relacionamentos ainda é a chave pra alguns no entendimento do que é um verdadeiro banco de dados.

AJAX

AJAX - Com o Ajax podemos fazer qualquer ação com o bando de dados no seu site sem precisar de um reload, ótimo se você não quer sua página carregando infinitas vezes não é?!

JavaScript

JavaScript - o JS é a maior linguagem web existente pela sua facilidade de usar, ficamos fascinados em sempre aprender novas coisas, é essencial umas boas interatividades com o usuário no seu site.

HTML

HTML -Linguagem de Sintaxe, obrigatória em todo sistema web.

CSS

CSS - Aqui, pra alguns desenvolvedores Front-End, é onde a mágica acontece, é aqui que visamos deixar o seu site estiloso, nossos projetos usam CCS3 pra trabalhar com os mais modernos estilos.

F.A.Q

P: Qual o preço do website completo?

Não colocamos preços permanentes que dependendo do serviço pode ser menor, por isso averiguamos com o cliente o tamanho do seu projeto.

P: Tem um prazo de tempo?

Para projetos de pequenos porte sim, de grande porte toda a equipe trabalha dobrado pra garantir o projeto mais rápido possivel.

P: Se eu não gostar, posso trocar?

Se completarmos o seu primeiro projeto, não tem mudança. Escolha com cautela, definimos bem o que será feito.

P: Se algo estiver errado no site, posso pedir pra ajeitarem?

Sim, nosso prazer é dar um projeto completo e sem problemas, sempre pode vim tirar dúvidas ou concertar algo que pode ter mudado na atualização das tecnologias.

P: Qual método de pagamento?

Usamos o método de pagamento MercadoPago, mas pra casos específicos e internacionais a cobrança é por Paypal.

P: Quanto custa hospedagem?

Não trabalhos com hospedagem, preço de hospedam não são inclusos, apenas da criação do website. Deixamos o site completo pra não precisar de grande configuração, porém preços de hospedam e de dominio é com empresas a parte, só o deixamos configurados.

P: Vocês trabalham com registro de dominio?

Não fazemos registro de dominio, caso seja necessário recomendamos e orientamos apenas.
